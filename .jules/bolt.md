## 2024-04-07 - Android TV Overdraw in Compose\n**Learning:** Composing multiple overlapping `Box` elements with backgrounds/gradients creates unnecessary node depth and can cause severe overdraw issues on low-end Android TV hardware.\n**Action:** Consolidate multiple static and dynamic gradient backgrounds into a single `Box` using `Modifier.drawWithCache` and `onDrawBehind` to draw multiple layers directly to the canvas, bypassing the Compose node hierarchy overhead for visual-only layers.
